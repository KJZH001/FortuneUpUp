# ✨简介✨

​	一个今日运势占卜工具，设计的初衷是为了取代博客在用的程序员老黄历，旨在给你的生活增添一些乐趣，设计启发和素材来自于《公主连结 Re:Dive》的一次新年占卜活动。

# 🪄使用🪄

添加以下代码到您的网页中

```html
<script src="FortuneUpUp.js"></script>
```

绑定事件

```html
<div id="StartFortuneUpUp" onclick="StartupFortuneUpUp()"></div>
```
